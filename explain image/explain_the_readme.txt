Merhaba Dostum!

Bu proje, BLIP (Bootstrapping Language-Image Pretraining) modelini kullanarak görüntüleri yorumlar başlık oluşturur. Model, video karelerindeki görsel içeriklere göre metinsel açıklamalar üretme yeteneğine sahiptir bu özelliği BLIP sağlıyor. Salesforce/blip-image-captioning-base önceden eğitilmiş modeli kullanarak her bir görüntü karesi  işlenir ve oluşturulan başlık gerçek zamanlı olarak videoya eklenir.

Kullanılan model hakkında daha fazla bilgi almak istersen " https://huggingface.co/docs/transformers/model_doc/blip " bağlantısına göz atabilirsin. 

	

Hello, my friend!

This project uses the BLIP (Bootstrapping Language-Image Pretraining) model to generate captions for images. The model has the ability to generate textual descriptions based on the visual content of video frames, and this feature is provided by BLIP. Using the pre-trained model Salesforce/blip-image-captioning-base, each video frame is processed, and the generated caption is added to the video in real-time.

For more information about the model used, you can check the following link: https://huggingface.co/docs/transformers/model_doc/blip.
